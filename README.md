# Dev Portal

A modern web portal for content and games with live tracking, chat, and suggestions.

## Features

- **Live User Tracking**: See who's online and what they're doing
- **Real-time Chat**: Communicate with other users
- **Content Suggestions**: Submit and vote on new content
- **Modern UI**: Responsive design with dark theme
- **Secure Firebase**: Environment variables protect API keys

## Deployment

Deployed on GitHub Pages with automatic GitHub Actions for secure Firebase configuration.
